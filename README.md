# 🚆 Train Delay Prediction System

A Machine Learning web application that predicts train delay probability using historical railway data.

🔗 **Live Demo:**  
https://train-delay-prediction-system.onrender.com

---

## 📌 Overview

This project uses a Random Forest model to estimate the probability of train delays based on route details, train type, weather conditions, and time factors.

The system provides a delay probability percentage instead of a simple delayed/on-time result.

---

## 🛠 Tech Stack

- Python
- Flask
- Scikit-learn
- Pandas & NumPy
- HTML & Tailwind CSS
- Render (Deployment)

---

## ⚙ How It Works

1. User enters train details.
2. Input is processed using trained encoders.
3. Random Forest model predicts delay probability.
4. Result is displayed on the web interface.

---

## ⚠ Disclaimer

This system provides predictions based on historical dataset patterns and does not use real-time railway data.
